# Cluster nodes

 Manual steps for odroid:
 * internet connection (wifi credentials)
 * start ssh
 * install docker armv7 
 * install kubernetes

 Manual steps for raspberrypi:
 * internet connection (wifi credentials)
 * enable & start ssh
 * install docker armv8
 * install kubernetes
